详细描述

设置和获取设备参数的万能接口，可以对相机参数节点表 中的参数进行获取和设置。

函数说明

◆ MV_CC_GetIntValue()

MV_CAMCTRL_API int __stdcall MV_CC_GetIntValue ( IN void *  handle,
IN const char *  strKey,
OUT MVCC_INTVALUE *  pIntValue 
)

获取Integer属性值

参数
handle[IN] 设备句柄
strKey[IN] 属性键值，如获取宽度信息则为"Width"
pIntValue[IN][OUT] 返回给调用者有关相机属性结构体指针
返回
成功，返回MV_OK；失败，返回错误码
备注
 • 连接设备之后调用该接口可以获取int类型的指定节点的值。strKey取值可以参考XML节点参数类型列表，表格里面数据类型为“IInteger”的节点值都可以通过该接口获取，strKey参数取值对应列表里面的“名称”一列。
示例
ChunkData.cpp, ConnectSpecCamera.cpp, GrabImage.cpp, GrabImage_HighPerformance.cpp, GrabMultipleCamera.cpp, ImageProcess.cpp, MultiCast.cpp, ReconnectDemo.cpp, SetParam.cpp , 以及 Trigger_Image.cpp.
